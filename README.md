# RYI (Raise Your Issues)
Web application that helps Issues

## Setup .env file

```
MONGO_URI=...
PORT=...
JWT_SECRET=...
NODE_ENV=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```
## Build the app
```
GRANT_TYPE=client_credentials
CLIENT_ID=client id
CLIENT_SECRET=client secret
```

## run server

```
npm run build
```
## Start the app

```
npm start
```
